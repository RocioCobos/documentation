Reservation

The Reservation operation requests a booking confirmation for a specific list of passengers. It provides the booking locator (booking code), which could be the supplier‚Äôs own code or the one sent in request, along with all the charges associated with the booking as well as its status. The returned fields include:

  • ProviderLocator
  • PropertyReservationNumber
  • ResStatus
  • Price

Status Code Description
