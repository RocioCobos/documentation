API Settings

In API Settings, you can easily view and manage your account settings. Just follow the steps below:

  1. Log onto TravelgateX

  2. Click on Connections > API Settings

  3. You will be able to find the following information:

    • Browsing as Buyer - You will find 2 tabs:

      • API Settings: it displays your API Key as Buyer, Business Settings and Timeout Settings. Both Business and Timeout settings displayed are the ones configured by default for your account, regardless of the folder, client and access - please note Settings set at request level override default settings.

      • Clients: here you are able to check the Hotel-X clients already created for your account, as well as generate new ones:

        • Adding new clients is as easy as clicking on "Create new client": When creating a new client, you'll be required to provide a client name. By default, all new clients will be granted full access. This means that if your Search query doesn't specify any access, and you only use the client, all accesses will be applied: If you wish to assign specific accesses to a particular client, you can make adjustments by selecting the "Edit default accesses" option: Simply uncheck the checkbox to grant the client full access by default, and then use the access boxes to choose only specific ones:

    • Browsing as Seller - You will find 1 tab:

      • API Settings: it displays your API Key as Seller.
info
  1. If you are using our Buyer Legacy API, your clients (or "hub users") have a password associated. You can check this password with the "Get password" option.
  2. Please be aware that you will have separate API Keys for your roles as a Seller and as a Buyer in our Platform. For example, if you are currently browsing as a Buyer and require access to your Seller API Key, you can easily switch your Partner profile type to Seller.
